The Remove link only appears if the Nanomaterials section appears (and there are nanomaterials in the draft).

The user is served a page/form showing all the nanomaterials (which are part of this draft) with checkboxes:

A Help/Hint text confirms that if all nanomaterials are removed then the user is required to complete task 1 again to add nanomaterials. (If all nanomaterials are deleted the nanomaterial quantity of task 1 is set to 0).

If the user has already completed the 'Category and formulation' task, an (!)Important help text statement confirms that if any nanomaterial is removed then the user is required to complete the 'Category and formulation' task again. (If a nanomaterial is removed, the 'Category and formulation' information is deleted and the 'Category and formulation' task is returned to its appropriate status - as does the final 'Accept and submit').

The page title is: "Select which nanomaterial(s) to remove".

A Cancel link allows the user to return to the draft page, but a green Continue button takes the user to a new page.

On the new page a Success banner confirms that the "nanomaterial(s) have been removed".

A green Continue button returns the user back to the draft page.

Return to previous draft state.
