The user completes this task, which includes providing a name - for the item in the multi-item kit - and follows the "Complete Product"/Categorise task journey.

  • At the end of the Categorise task journey the user has the option to:
    1. Continue directly onto the next item (if there is one)
    2. Or return to the drafts page
  • When following the items task journey, we add the item title to the H1 title of each page (starting with its holding title)
  • When returning to the draft page, the corresponding task in the 'Category and formation' section has updated its name. (The two are linked).

After completing the task the user returns to the drafts page.
