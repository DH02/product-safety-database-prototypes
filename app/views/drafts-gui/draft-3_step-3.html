The user completes the Nanomaterials task (completing its details, including its name).

  • When following the Nanomaterials task journey, we add the item title to the H1 title of each page (starting with its holding title)
  • At the end of the Nanomaterials task/journey, the user flow offers the option to continue straight onto the next one (if there is another) or return back to the draft page

For this click-through demo, we assume that all nanomaterials have been completed, when the user returns to the drafts page.
