Search behaviour - notes:

Submitting the keywords refreshes the search results: if the filters have been set, the results incorporate the filter settings and the filters form maintains state (if the page is refreshed during the search). The span (id, opss-filters-txt) is also included in the results overview text.

If the user selects the “clear filters” link (coded as a form button), this refreshes the page results, returns the filters form back to its original state (with radios collapsed) and does not include the span (id, opss-filters-txt) in the results overview text.
